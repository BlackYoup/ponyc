class builtin/String is Seq[U8 val] ref, Comparable[String box] ref, Stringable[FormatDefault val, PrefixDefault val] box

Default capability: val

Strings don't specify an encoding.

Constructors

new copy_cstring(str: Pointer[U8 val] box, optional len: USize val)

If the cstring is not null terminated and a length isn't specified, this can crash. This will only occur if the C-FFI has been used to craft such a pointer.

new create(optional len: USize val)

An empty string. Enough space for len bytes is reserved.

new from_array(data: Array[U8 val] val)

Create a string from an array, reusing the underlying data pointer if the array is null terminated, or copying the data if it is not.

new from_cstring(str: Pointer[U8 val] ref, optional len: USize val)

The cstring is not copied. This must be done only with C-FFI functions that return null-terminated pony_alloc'd character arrays.

new from_utf32(value: U32 val)

Create a UTF-8 string from a single UTF-32 code point.

Read an integer from the specified location in this string. The integer value read and the number of characters consumed are reported. The base parameter specifies the base to use, 0 indicates using the prefix, if any, to detect base 2, 10 or 16.
